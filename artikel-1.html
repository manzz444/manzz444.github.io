🚀 Week 1: The First HTML

📅 15 Februariy2026 ⏱️ 3 minute read

This is my first time actually touching coding. At first, I was confused about the tags, but after trying it out in VS Code, I began to understand the basic structure of HTML. I was so happy when I saw the text for the first time. "Hi, my name is Ilham" appears in the browser.

📝 The first HTML structure created:

<!DOCTYPE html>
<html>
<head>
    <title>My First Website</title>
</head>
<body>
    <h1>Hi, my name is Ilham</h1>
    <p>I am learning to code</p>
</body>
</html>

🔥 New things learned:

  • HTML tag: h1, p, a, img, ul, li
  • Document structure: html, head, body
  • Attribute: src, href, alt, class
  • VS Code: install, create a file, save
  • GitHub: bikin akun, upload file

💡 Important takeaway:

Don't be afraid of making mistakes, just get started.. Coding is like playing a game: the more you try, the more you understand. The vibe of coding with AI is real!
