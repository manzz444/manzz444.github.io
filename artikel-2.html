💡 Minggu 2: Belajar CSS & Bikin Desain

📅 22 Februari 2026 ⏱️ 6 menit baca

Setelah HTML jadi, waktunya bikin website gak cuma putih polos. Belajar CSS dari nol: padding, margin, color, flexbox, grid. Pusing sih, tapi tiap kali lihat hasil berubah langsung puas. Dark mode juga berhasil ditambahin!

🎨 Yang dipelajari di CSS:

  • Selectors: element, class, id
  • Box Model: margin, padding, border
  • Colors: hex, rgb, gradient
  • Flexbox: display: flex, justify-content, align-items
  • Grid: grid-template-columns, gap
  • Responsive: media query untuk mobile
  • Animasi: transition, transform

🔄 Proses belajar:

  1. Mulai dari ngubah warna background
  2. Bikin navigasi pake flexbox
  3. Atur card project pake grid
  4. Bikin dark mode toggle pake JavaScript
  5. Tambahin animasi hover
  6. Bikin responsive buat HP

🔥 Momen seru:

Pas pertama kali berhasil bikin card project pake grid dan liat hasilnya rapi, rasanya puas banget! Dark mode juga jadi fitur favorit, tinggal klik tombol, website berubah tema.
