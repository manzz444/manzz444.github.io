💡 Week 2: Learning CSS & Creating Designs

📅 22 February 2026 ⏱️ 6 minutes read

Once the HTML is done, it's time to make the website more than just plain white. Learn CSS from scratch: padding, margin, color, flexbox, grid. It's confusing, but every time I see the results change, I'm immediately satisfied. Dark mode has also been successfully added.!

🎨 What i'm studied in CSS:

  • Selectors: element, class, id
  • Box Model: margin, padding, border
  • Colors: hex, rgb, gradient
  • Flexbox: display: flex, justify-content, align-items
  • Grid: grid-template-columns, gap
  • Responsive: media query untuk mobile
  • Animation: transition, transform

🔄 Learning process:

  1. Start by changing the background color
  2. Create navigation using flexbox
  3. Arrange project cards using grid
  4. Create a dark mode toggle using JavaScript
  5. Add hover animation
  6. Make it responsive for mobile phones

🔥 Exciting moments:

The first time I successfully created a card project using a grid and saw the neat results, I felt so satisfied! Dark mode is also a favorite feature. Just click the button, and the website changes its theme.
