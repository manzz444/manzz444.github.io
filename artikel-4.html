🚀 Minggu 1: HTML Pertama dalam Hidup

📅 15 Februari 2026 ⏱️ 3 menit baca

Ini pertama kalinya nyentuh coding beneran. Awalnya bingung bedain tag, tapi setelah nyoba-nyoba di VS Code, mulai paham struktur dasar HTML. Seneng banget pas pertama kali lihat tulisan "Hi, my name is Manzz" muncul di browser.

📝 Struktur HTML pertama yang dibuat:

<!DOCTYPE html>
<html>
<head>
    <title>Website Pertamaku</title>
</head>
<body>
    <h1>Hi, my name is Manzz</h1>
    <p>I am learning to code</p>
</body>
</html>

🔥 Hal-hal baru yang dipelajari:

  • Tag HTML: h1, p, a, img, ul, li
  • Struktur dokumen: html, head, body
  • Attribut: src, href, alt, class
  • VS Code: install, bikin file, save
  • GitHub: bikin akun, upload file

💡 Takeaway penting:

Gak perlu takut salah, yang penting mulai aja dulu. Coding itu kayak main game: makin sering dicoba, makin paham. Vibe coding with AI is real!

Waktu itu masih bingung kenapa harus pake <!DOCTYPE html> dan apa bedanya <head> sama <body>. Tapi setelah beberapa kali latihan, akhirnya paham juga.
