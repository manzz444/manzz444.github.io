🎉 Week 4: FINALLY APPEARING ON GOOGLE!

📅 8 March 2026 ⏱️ 4 minutes read

I never expected that! After submitting the sitemap and requesting indexing, about 5 days later the website started appearing on Google. When I checked using site:manzz444.github.io, The results came out. I felt a mixture of happiness and disbelief. It really worked!

📊 The process:

  • Days 1-2: Submit your sitemap to Google Search Console
  • Day 3: Request indexing via the URL Inspection feature
  • Day 5: Check using "site:" and it appears! 🎉

🔥 What I've learned:

  • Basic HTML & CSS
  • Google Analytics (installing tracking code)
  • Google Search Console (verification + sitemap)
  • Dark mode toggle using JavaScript
  • Contact form using Formspree
  • And most importantly: patience

💡 Tips for beginners:

Don't be afraid to make mistakes. Ask AI (ChatGPT/Deepseek) for help if you're confused. The important thing is to get started first, tidy up later. The coding vibe is real!
