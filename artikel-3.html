🔧 Minggu 3: Setup Google Search Console & Sitemap

📅 1 Maret 2026 ⏱️ 5 menit baca

Minggu ini agak pusing karena urusan verifikasi website ke Google. Harus nambah meta tag, upload sitemap.xml, dan minta Google buat crawl. Tapi ternyata gak serumit yang dibayangin. Tinggal ikutin langkah-langkah dari AI, beres.

📝 Langkah-langkahnya:

  1. Daftar di Google Search Console
  2. Pilih metode verifikasi "HTML tag"
  3. Copy meta tag, tempel di <head> website
  4. Upload ulang ke GitHub
  5. Klik Verify di Search Console
  6. Buat file sitemap.xml dan upload
  7. Submit sitemap lewat dashboard

💡 Tips penting:

Jangan lupa daftarin sitemap di Google Search Console biar Google cepet nemuin halaman kita. Sitemap itu kayak daftar isi website, bikin Google lebih gampang ngindex.

Meta tag verifikasi juga jangan sampe kehapus. Simpen baik-baik di file HTML.
