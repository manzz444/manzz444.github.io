🔧 Week 3: Set up Google Search Console & Sitemap

📅 1 March 2026 ⏱️ 5 minutes read

This week has been a bit stressful because of website verification issues with Google. I had to add meta tags, upload sitemap.xml, and ask Google to crawl. But it turned out to be not as complicated as I had imagined. I just followed the steps from AI, and it was done.

📝 The steps:

  1. Sign up for Google Search Console
  2. Select the "HTML tag" verification method
  3. Copy the meta tag and paste it into the <head> section of your website
  4. Re-upload to GitHub
  5. Click Verify in Search Console
  6. Create a sitemap.xml file and upload it
  7. Submit the sitemap via the dashboard

💡 Important Tips:

Don't forget to register your sitemap. in Google Search Console so that Google can quickly find our pages. A sitemap is like a table of contents for a website, making it easier for Google to index it..

Don't delete the verification meta tag either. Keep it safe in the HTML file..
