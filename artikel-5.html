🎮 Experimental Themes

Ini yang paling seru. Aku bikin 4 tema berbeda yang bisa diakses lewat halaman utama. Masing-masing punya gaya dan fitur unik.

💻 Hacker Terminal

Tema dengan nuasa terminal hacker. Background hitam, teks hijau neon, font monospace, dan navigasi ala command line. Cocok buat yang suka estetika cyber.

Fitur: ASCII banner, command-style menu, status sistem online.

🕹️ Retro Games

Nuansa gameboy / retro game. Background gelap dengan aksen warna cerah, font pixel, dan tombol ala konsol. Skill ditampilkan sebagai HP, EXP, MP.

Fitur: Screen ala gameboy, tombol START, skor dan hi-score.

🎮 Tribute Page

Halaman tribute buat game-game legenda. Desainnya terinspirasi dari era 8-bit, lengkap dengan efek CRT (tabung TV jadul).

Fitur: Efek scanline, pixel art, stat ala game (ATK, DEF, HP).

📊 Hacker Dashboard

Dashboard ala admin hacker. Menampilkan statistik pengunjung, progress belajar, dan log aktivitas. Cocok buat yang suka tampilan data.

Fitur: Stats grid, progress bar, log real-time, menu navigasi.

📁 Struktur Folder Baru

Biar makin rapi, semua file tema eksperimental dipisah dalam folder sendiri:

themes/
├── hacker/
├── games/
├── tribute/
└── dashboard/

Masing-masing punya index.html dan style.css sendiri. Gak campur aduk sama file utama.

💡 Yang Dipelajari Minggu Ini

  • Bikin struktur folder yang rapi dan scalable
  • Desain dengan tema berbeda (hacker, retro, glassmorphism)
  • Local Storage di JavaScript
  • Navigasi antar halaman dengan link relatif
  • CSS murni tanpa framework (tapi tetep kece)
